What is Seattle Board Game Corner?

Seattle Board Game Corner is a mobile-friendly web application that allows board game lovers to learn about the current hot board games, trade their board game collections with other users'.

Users can post their collections of board games on their pages. Users can also search a certain board game by name and find out which user in the Seattle area has the board game. Simply by sending a message to the game owner, users can start trade their board games with other users' collections and get to play different board games without buying the game.
